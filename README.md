# SAML2 Service Provider

This is a fork of a fork with a lot of own improvements based on [24Slides/laravel-saml2](https://github.com/24Slides/laravel-saml2) which turns your laravel application into Service Provider via SAML2 protocol based on [OneLogin](https://github.com/onelogin/php-saml) toolkit.


Please refeer to the repository of 24Slides to find basic information, documentation and so on. Currently I'll focus on development for an updated secure version of this package (and will not focus on documentaion :P)
